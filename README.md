# DINI

DINI (DINI Is Not IMDB) is an experimental in-memory document storage engine designed for DiffScope. It exposes a pure C++ handle-based API for schema definition, transactions, ordered association-grouped lists, parent relations, polymorphic columns, query views, change sets, snapshots, commit logs, and in-memory undo/redo history.

> The "IMDB" here stands for "in-memory database", not "Internet Movie Database"
